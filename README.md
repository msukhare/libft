# LIBFT

## ABOUT LIBFT
* Libft is a static library.

## What you need to make libft

* gcc or clang

* [ar](https://linux.die.net/man/1/ar)

## COMPILE LIBFT

* Use make to generate `libft.a`.

Makefile has 3 rules:

  * `all`: create libft.a
  
  * `clean`: delete all objects
  
  * `fclean`: delete libft.a and clean
